Installing and Uninstalling Carbide.V

Checking That Your System is Compatible with Carbide.v

Current minimum system requirements of Carbide.v are:
  • 200 MB hard disc space for installation
  • 512 MB free memory for runtime
  • Windows XP or Windows Vista operating system
  • 5-50 MB temporary free hard disc space for each opened CPF

Obtaining Latest Version of Carbide.v

Current version of Carbide.v is 2.0 Beta. It is not distributed via Forum Nokia yet. You can obtain the latest release installation file via Carbide.v Nokia S60 wiki page. Download the installer executable to your disc, and proceed to the next step.

Installing of Carbide.V

After you have got the Carbide.v installer file, you simply launch it.

  1. Launch Carbide.v installer by double-clicking Carbide.v setup icon or file Carbide.v installer opens.
  2. Click Next to proceed.
  3. Read Carbide.v license terms.
    • Choose I accept the terms of the license agreement and click Next to proceed with installation.
    • Choose I do not accept the terms of the license agreement and click Cancel if you want to stop the installation.
  4. Select the folder in which to install Carbide.v. Click Change to change the default installation folder. After choosing the installation folder, click Next to proceed.
  5. Select the program folder in which Carbide.v appears in your Start Menu. By default, Carbide.v will be installed in its own folder. Click Next to proceed.
  6. Start installation. Carbide.v now has all the information needed for installing. Click Install to start installing Carbide.v in its installation folder. The installation may take up to a few minutes.

After these steps, Carbide.v is installed on your computer. You may select to view the Release Note or to open Carbide.V. Click Finish to exit the installer.

Do not delete you Carbide.v installer. You may need it later on for repairing a possibly broken Carbide.v installation.

Uninstalling Carbide.V

You can uninstall Carbide.v in two ways: Either through Control Panel > Add or Remove Programs or through Carbide.v installer. If you uninstall Carbide.v through Control Panel, you may skip the first two steps below.
  1. Open Carbide.v installer by double-clicking the Carbide.v setup icon or file. Carbide.v installer opens.
  2. Select Remove to choose uninstalling Carbide.v. Click Next to proceed.
  3. Installer confirms that you want to uninstall Carbide.v. Click No to return to previous dialog. Click Yes to uninstall Carbide.v. The installer will remove Carbide.v. After installation is done, click Finish to exit installer.

After these steps, Carbide.v is uninstalled from your computer. If you wish, you can now install Carbide.v.

Upgrading Installation

Situation: You have got a new version of Carbide.v as an installer file.
  1. Open Carbide.v installer by double-clicking Carbide.v setup icon or file. Carbide.v installer opens.
  2. Select Repair to upgrade the installation. Click Next to proceed. Installer upgrades Carbide.v. Click Finish to exit installer.

Repairing Broken Installation

Carbide.v may get accidentally broken by a third party application or software error. Carbide.v installer can repair the broken sofware.
  1. Open Carbide.v installer by clicking Carbide.v setup icon or file. Carbide.v installer opens.
  2. Select Repair to repair broken installation. Click Next to proceed. Installer will replace broken Carbide.v files. After repair is done, click Finish to exit installer.
