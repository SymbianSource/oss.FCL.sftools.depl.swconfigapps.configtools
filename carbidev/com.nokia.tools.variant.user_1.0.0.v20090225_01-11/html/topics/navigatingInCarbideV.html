Navigating in Carbide.v

An S60 phone software configuration includes much information that is not relevant in filling out a variant proposal. Carbide.v shows you a pre-organized view to the CPF: The main category and setting groups. they assist you in navigating to the relevant customizable settings.

Menu content

This part describes all the menu options of the application.
  1. File option.
    Open - opens a standard Open dialog..
    Open As New - opens a standard Open dialog. CPF is opened as untitled.
    Close - closes the active CPF..
    Close all - closes all CPFs that are currently open in the tool. Disabled if there is just one CPF open in the tool.
    Save - saves the active CPF.
    Save as - opens a standard Save As dialog.
    Exit - ends the session and closes the whole appication.

  2. Edit option.
    Undo - undoes an action user has performed. Note: Undo shall be unlimited i.e. saving should not erase the undo memory.
    Redo - redoes an action that user had undoed, or repeats an action that user last performed.
    Cut - cuts a string or resource to the clipboard.
    Copy - copies a string or resource to the clipboard.
    Paste - pastes a string or resource from the clipboard to a field that currently has focus.
    Delete - clears a string or resource from a field.
    Select all - selects all fields in the active form.
    Hide Selected Settings - removes the selected settings from the active CPF file. Disabled when no settings are selected.
    Reset to Default Values - resets the selected (modified) settings back to their default values. Disabled when no settings are selected.

  3. Tools option.
    Copy values - opens the Copy Values dialog with the active CPF as a target.
    Create report - opens the Create Report dialog..

  4. View option.
    Properties - displays / hides the Properties view.
    Preview - displays / hides the Preview view.
    Resources - displays / hides the Resources view.
    Errors - displays / hides the Errors view.
    Notes - displays / hides the Notes view.
    Changes - displays / hides the Changes view.
    Search results - displays / hides the Search Results view.
    Toolbar - displays / hides the toolbar.
    Status bar - displays / hides the statusbar.
    Reset Layout - resets the workspace to the default panel layout.
    Window - opens a submenu.
    • New - opens a new empty tab to the workspace.
    • 1-X (cpf name) - displays all open CPFs in the list, the currently active CPF has a checkmark..

  5. Help option.
    Carbide.v Help - opens a Help dialog.
    Check For Updates - opens a dialog for checking updates.
    Report Bugs - opens a dialog in which the user can send a free text bug report.
    About Carbide.v - opens About Carbide.v popup dialog.

Navigating By Groups

Situation: You want to view the settings in logically organized groups. A phone software configuration may include a lot of information that is not relevant in variant customization, from the end users' point of view. Therefore, CPFs are customized to show you only the settings that are relevant to you. These settings are organized into meaningful groups that Carbide.v displays you in its navigator. The navigator displays you main category and setting groups that assist you in finding the relevant customizable settings.
  1. Select the main category (the large icons on the left).
  2. Carbide.v lists the setting group titles beside the category icon.
  3. Select the setting group of your interest.
Carbide.v opens the selected group of settings in the software proposal form.
If you did not find the setting you were looking for, use the Search.
