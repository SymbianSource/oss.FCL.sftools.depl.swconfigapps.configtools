Create new files into the layers

For creating different types of files there are wizards in the tool to ease the job. There are two main wizards: wizard for creating ConfML files and wizard for creatin ImplML files. The ConfML wizard is able to create ConfML files with initial content. The ImplML wizard is able to create all kinds of other ML files, which are supported, including .crml, .gcfml, .hcrml, .implml, .ruleml, .content, and more. To create new files with wizards follow these steps:

  1. Right-click the folder where you want to create the file (usually confml folder in the layer).
  2. Select New.
  3. From the popup menu choose either Configuration File or ImplML file
  4. Let the wizard guide you the rest of creating the new file
