Order of tasks in using Carbide.ct

  1. To install, download the zip package and extract it to the folder of your choice.
  2. Create a new Configuration Project.
  3. Create one or more layers for the project.
  4. Import files into the layer.
  5. Create new files into the layers.
  6. Add layer and configuration root files to the Configuration Project.
  7. Select active Configuration root file.
  8. Refine values with value editor.
  9. Define new View definitions for grouping of ConfML features and settings.
